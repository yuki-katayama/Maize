# Maize
## 日本語
### 概要
迷路の中を進んで、ゴールを目指すゲームです

### ルール
+ 針を踏むと、ゲームオーバー
+ ゴールの下を通れば、クリアです。

## English
### Summary
This is a game where you go through a maze to get to the goal!

### Rules.
+ If you step on the needle, the game is over.
+ If you pass under the goal, you are in the clear.
